📊 E-Commerce Sales Dashboard (Excel)

🔹 Project Overview

This project presents an interactive Excel dashboard designed to analyze sales, profit, and customer behavior in an e-commerce business.

🔹 Key Features

* KPI Metrics (Sales, Profit, Quantity, Orders, Profit Margin)
* Category-wise Profit Analysis
* Monthly Sales Trends
* State-wise Sales Visualization
* Top 5 Products Analysis
* Interactive Filters (Month, Region, Segment)

🔹 Key Insights

* Sales increased by 12.46% YoY, but profit decreased by 1.72%
* Profit margin dropped significantly, indicating cost inefficiencies
* Technology category contributes highest profit
* Chairs and Phones are top-performing products
* Sales peak during festive months (Sep–Nov)
